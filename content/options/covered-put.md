# Covered Put 是什么？

Covered Put 通常指做空股票，同时卖出 Put 收取权利金。它适合偏空或中性偏空观点，但风险并不低，因为做空股票理论上有无限亏损风险。

策略逻辑：

- 股票下跌或横盘时，卖 Put 的权利金可以增加收益。
- 如果股票跌破 Put 执行价，空头 Put 可能被指派，用较低价格买回股票平掉部分空头。
- 如果股票大涨，做空股票亏损可能远超收到的权利金。

交易前必须检查：

1. 做空股票是否容易借到，融券费多少。
2. 最大可承受亏损是多少。
3. Put 的执行价是否对应愿意回补的位置。
4. 是否存在财报、并购、逼空等跳空风险。
5. 保证金是否足够应对股价反向大涨。

新手通常不适合直接使用 Covered Put。它比 Covered Call 更难管理，因为上涨风险没有天然上限。
