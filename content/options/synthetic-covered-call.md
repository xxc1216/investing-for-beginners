# 合成 Covered Call 是什么？

合成 Covered Call 通常用期权组合复制持股卖 Call 的收益结构。例如通过卖 Put 等方式构造类似暴露。

它的重点不是名字，而是风险是否真的等价。

检查清单：

1. 最大亏损是否和持股一致。
2. 是否需要保证金。
3. 是否存在提前指派。
4. 股息和除息风险如何处理。
5. 流动性是否足够。

新手不要因为“合成”听起来更高级就使用。先能完整解释普通 Covered Call，再看合成结构是否有实际优势。
