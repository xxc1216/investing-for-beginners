# 共识机制解决什么问题？

共识机制是区块链用来让网络参与者对交易顺序和账本状态达成一致的规则。没有共识机制，去中心化网络就无法判断哪条记录有效。

常见类型：

- PoW：通过算力竞争出块，比特币代表。
- PoS：通过质押资产参与验证，以太坊等链使用。
- BFT 类机制：常见于部分高性能链和联盟链设计。

分析共识机制时看四点：

1. 攻击成本是什么。
2. 验证者或矿工是否集中。
3. 出块和最终性如何实现。
4. 作恶惩罚和激励是否清晰。

新手不需要一开始研究所有数学细节，但要知道：共识机制决定了链的安全边界、去中心化程度和极端情况下的恢复能力。
