# Mempool 交易替换是什么？

Mempool 交易替换指未确认交易还在内存池中时，用户用更高费用的新交易替代旧交易。它常用于加速卡住的交易或取消未确认交易。

常见场景：

- Gas 设太低，交易迟迟不确认。
- 想加速同一笔转账。
- 想用同一 Nonce 发送 0 金额交易来取消。

新手要理解 Nonce：同一地址的交易通常按 Nonce 顺序执行。低 Nonce 交易卡住，后面的交易也可能排队。

操作前注意：

1. 新交易必须使用相同 Nonce。
2. 手续费要足够高，才能被节点接受替换。
3. 交易已经确认后无法取消。
4. 不同钱包的“加速/取消”按钮实现略有差异。
