# 空投钱包为什么要隔离？

空投交互常常需要连接陌生网站、签名、授权和测试新合约。钱包隔离的目的，是把高风险操作限制在小账户里，避免主钱包资产被一次恶意授权拖走。

建议分四层：

1. 主钱包：长期资产，不连接陌生网站。
2. 交易钱包：常规 DEX、CEX 提现和链上操作。
3. 空投钱包：测试网、新项目、任务和签名交互。
4. 废弃钱包：曾经暴露高风险授权，不再存放重要资产。

每次交互前检查：域名是否官方、合约地址是否可核验、是否要求无限授权、签名内容是否能读懂、损失是否低于该钱包预算。

交互后要记录项目、日期、链、交易哈希和成本；撤销不再需要的授权；把多余资产转回安全钱包；可疑钱包直接废弃。
