# DeFi 清算缓冲应该怎样设置？

DeFi 借贷仓位不能只看“还没到清算线”。清算缓冲是你在价格跳空、网络拥堵、预言机延迟和利率上升时，仍能处理仓位的安全距离。

先记录仓位参数：

1. 抵押资产数量和价格。
2. 借入资产数量和价格。
3. 最大 LTV、清算阈值和清算罚金。
4. 当前健康因子。
5. 可立即还款或补仓的资产。

再设三条线：

- 正常目标线：不需要操作，但持续监控。
- 预警线：减少杠杆、补充抵押或准备还款。
- 强制降杠杆线：无论观点如何，先降低清算风险。

补抵押不一定比还款更好。补抵押会增加风险资产暴露；还款能直接降低债务，但需要可用稳定资产。

新手最好避免把健康因子压得太低。链上极端行情中，交易所提现、钱包交易和协议前端都可能同时变慢。
