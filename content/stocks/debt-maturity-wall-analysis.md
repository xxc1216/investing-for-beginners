# 债务到期墙应该怎样分析？

债务到期墙指公司未来几年集中到期的债务本金。它比“总负债多少”更重要，因为风险往往发生在现金流不足、利率上升、信用市场关闭时。

分析步骤：

1. 从 10-K、10-Q 和债务附注中找出现金、短债、长期债务当期部分、循环授信和债券到期表。
2. 按年份列出未来五年要还的本金。
3. 加入租赁、采购承诺等固定现金支出。
4. 估算每年可用资金：现金、自由现金流、未使用授信、可出售资产。
5. 计算缺口：到期债务加固定承诺，减去可用资金。

重点不是公司能不能“理论上再融资”，而是在压力情景下还能不能再融资。若评级下调、利润下滑、利率上升同时发生，到期墙会迅速变成流动性危机。

新手可以给每家公司做三种情景：基准、衰退、信用关闭。只要信用关闭情景下资金缺口很大，仓位就不该太激进。
