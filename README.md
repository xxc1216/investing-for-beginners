小隐寺面向中文美股与加密货币投资者，提供结构化的基础知识、交易概念与风险教育。

这个仓库是小隐寺投资百科的官方公开知识库，主要收录：

- 美股基础知识；
- 期权概念与交易风险；
- 加密货币基础知识；
- 小隐寺投资百科的精选内容与官方入口。

> 本仓库不提供个性化投资建议，也不承诺任何收益。金融产品存在本金损失风险，使用杠杆和衍生品可能放大亏损。
